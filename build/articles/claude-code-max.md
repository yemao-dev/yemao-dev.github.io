## 快速接入（Quick Start）

以 **Claude Code** 终端为例，只需简单两步：

### 1. 设置环境变量

在你的终端中执行以下命令（建议写入 `~/.zshrc` 或 `~/.bashrc`）：

```bash
export ANTHROPIC_BASE_URL="https://www.kingflow.ai"
export ANTHROPIC_AUTH_TOKEN="你的API-KEY"
```

### 2. 启动客户端

```bash
# 安装 / 更新
npm install -g @anthropic-ai/claude-code

# 开启
claude
```
